# Coding Decisions

Conventions and coding patterns the agents should follow, derived from the
existing codebase and Laravel Boost project instructions.

- Follow PSR-12 / Laravel Pint for PHP.
- Use the Composition API and `<script setup>` for Vue components.
- Prefer strict TypeScript; avoid untyped `any`.
- Use Conventional Commits.
- Reuse existing services instead of introducing new abstractions.
